🎉 WHAT'S NEW IN v1.1.0
================================================================================

This release includes the complete suite of Windows system discovery, audit,
and deployment tools built with Go.

NEW IN THIS RELEASE:
--------------------
★ Go Diff Packer - Directory comparison and deployment tool
  • Compare two directories for changed/new files
  • Size-first comparison with SHA-256 hash verification
  • JSON output mode for AI/automation integration
  • Interactive deploy mode with replace confirmation
  • Auto-incrementing output folders (_outdiff_01, _outdiff_02, etc.)

================================================================================
📦 INCLUDED TOOLS
================================================================================

| Tool                        | Description                                |
|-----------------------------|--------------------------------------------|
| system_info.exe             | Memory, CPU, Storage, OS, Security, Network|
| xampp_collector.exe         | XAMPP discovery and version auditing       |
| patch_collector.exe         | Windows Update/HotFix collector            |
| firewall_collector.exe      | Windows Firewall rules auditor             |
| archiver.exe                | Unicode-aware ZIP archiver (Thai, Chinese) |
| unarchiver.exe              | Interactive ZIP extractor                  |
| db_manager.exe              | MariaDB backup, restore, and audit         |
| db_mcp_server.exe           | Database MCP server for AI assistants      |
| dev_tool_collector.exe      | Dev tools discovery (Go, Rust, Python...)  |
| wsl_manager.exe             | WSL management utility                     |
| go-diff-packer.exe          | Directory diff and deployment tool [NEW!]  |

================================================================================
📥 INSTALLATION
================================================================================

1. Download all .exe files from this release
2. Place them in a folder (e.g., C:\tools\win-devops-toolkit-2026)
3. Run any tool directly - no installation required!

All tools are portable executables with no dependencies.

================================================================================
🔧 REQUIREMENTS
================================================================================

• Windows 10/11 or Windows Server 2016+
• Go 1.21.1+ (only if building from source)
• Administrator privileges (for some tools)

================================================================================
📖 QUICK START GUIDE
================================================================================

System Audit:
    .\system_info.exe

XAMPP Discovery:
    .\xampp_collector.exe

Directory Comparison:
    .\go-diff-packer.exe C:\original C:\modified

Directory Comparison (JSON output):
    .\go-diff-packer.exe -json C:\v1 C:\v2 > changes.json

Interactive Deploy:
    .\go-diff-packer.exe --deploy

Database Backup:
    .\db_manager.exe backup

Dev Tools Audit:
    .\dev_tool_collector.exe
