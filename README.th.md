## 🚀 คุณสมบัติหลัก

### 1. เครื่องมือรวบรวมข้อมูลระบบ (`system_info.exe`)
- **หน่วยความจำ (Memory):** แสดงค่า RAM ที่แม่นยำ (ทั้งหมด, ว่าง, อัตราการใช้งาน) ผ่าน Windows `syscall`
- **หน่วยประมวลผล (CPU):** รายละเอียดสเปก CPU และจำนวนคอร์
- **พื้นที่จัดเก็บข้อมูล (Storage):** วิเคราะห์ Logical Disk (พื้นที่ทั้งหมด/ว่าง/ที่ใช้ไป, ระบบไฟล์)
- **ระบบปฏิบัติการ (OS):** ระบุเวอร์ชันและเลข Build ของ Windows
- **ความปลอดภัย (Security):** ตรวจสอบสถานะ Antivirus และ Firewall ที่ใช้งานอยู่ผ่าน WMI
- **เครือข่าย (Network):** การตั้งค่า Adapter ทั้งหมด รวมถึง IP Address และ DNS Search Order

### 2. เครื่องมือสแกนระบบ XAMPP (`xampp_collector.exe`)
- **การค้นหา (Discovery):** ตรวจพบโฟลเดอร์ `xampp*` โดยอัตโนมัติในไดรฟ์ `C:`, `D:`, และ `E:`
- **การคำนวณขนาด (Sizing):** คำนวณพื้นที่ที่ XAMPP ใช้ทั้งหมด และขนาดของโฟลเดอร์ `htdocs` โดยเฉพาะ
- **การตรวจสอบเวอร์ชัน (Version Auditing):** ดึงข้อมูลเวอร์ชันที่แน่นอนของ **PHP** และ **MariaDB/MySQL** โดยการคิวรีจากไฟล์ Binary โดยตรง
- **การแสดงผล (Output):** บันทึกผลลัพธ์ในรูปแบบ JSON และ TXT (อ่านง่าย) ไว้ในโฟลเดอร์ `./output/`

### 3. เครื่องมือตรวจสอบแพตช์ Windows (`patch_collector.exe`)
- **การตรวจสอบ:** ดึงข้อมูลรายการอัปเดตและ HotFix ทั้งหมดที่ติดตั้งในระบบผ่าน PowerShell `Get-HotFix`
- **รายละเอียด:** แสดง HotFixID (หมายเลข KB), คำอธิบาย, วันที่ติดตั้ง และผู้ติดตั้ง
- **การแสดงผล:** บันทึกข้อมูลในรูปแบบ JSON และ TXT เพื่อใช้ในการตรวจสอบความปลอดภัย

## 🛠️ วิธีการใช้งาน

### การคอมไพล์ (Build)
หากต้องการสร้างไฟล์โปรแกรม (.exe) จากซอร์สโค้ด:
```powershell
.\build.bat
```

### การเรียกใช้งาน (Run)
1. รัน `system_info.exe` เพื่อตรวจสอบข้อมูลระบบทั่วไป
2. รัน `xampp_collector.exe` เพื่อค้นหาและตรวจสอบเว็บเซิร์ฟเวอร์
3. ตรวจสอบรายงานละเอียดได้ในโฟลเดอร์ `./output/`
